抖音信息提取器：智能识别抖音主页类型，自动获取并展示博主昵称、SecUID、UID、SecUserID 以及 Cookie 信息，并支持一键复制功能。

使用说明：
本脚本需搭配浏览器扩展程序「篡改猴 (Tampermonkey)」使用：https://www.tampermonkey.net/

请先安装 Tampermonkey 后再导入脚本，否则将无法正常运行。

本插件的部分设计理念与功能参考了抖小云作者的优秀项目，在此表达诚挚感谢。
dysync.net - 抖音同步工具（抖小云）：https://github.com/jianzhichu/dysync.net

dysync.net 基于 .NET Core 与 Vue 开发，可用于同步抖音收藏夹、“我喜欢”的视频及指定博主的作品，并支持多账号。工具内置视频信息刮削功能，同步完成后的视频可直接在 Emby 或 Jellyfin 中播放。

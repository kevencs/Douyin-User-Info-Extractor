# 抖音信息提取器 (Douyin User Info Extractor) v10.2

## 项目简介
抖音信息提取器是一款功能强大的浏览器用户脚本，能够智能识别抖音主页类型，自动获取并展示博主昵称、SecUID、UID、SecUserID等关键信息，并提供便捷的一键复制功能。脚本采用现代化UI设计，操作流畅，界面美观。

> **注意**：由于浏览器安全限制，完整的Cookie信息需要手动通过开发者工具获取。

## 功能特性
- 🔍 **智能识别**：自动区分"自己主页"与"他人主页"，采用不同配色方案
- 📊 **数据提取**：实时提取博主昵称、SecUID、UID、SecUserID等关键信息
- 📋 **一键复制**：支持单个信息复制或全部信息一键导出
- 🎨 **精美界面**：现代化渐变UI设计，悬浮按钮，响应式布局
- ⚡ **自动获取**：通过监听网络请求智能获取用户信息
- 📱 **移动适配**：完美适配桌面端和移动端浏览器

## 安装使用

### 前提条件
本脚本需要配合浏览器扩展程序「篡改猴 (Tampermonkey)」使用。

**安装步骤：**
1. 首先安装 [Tampermonkey 扩展](https://www.tampermonkey.net/)
2. 点击脚本链接导入到 Tampermonkey 中
3. 启用脚本后访问抖音网页版即可使用

### 使用说明
1. **打开抖音网页版**：访问 https://www.douyin.com/
2. **浏览用户主页**：访问任意用户主页或自己的主页
3. **点击悬浮按钮**：页面右下角会出现悬浮按钮，点击即可打开信息面板
4. **查看提取结果**：面板中将显示已提取的用户信息
5. **一键复制**：点击对应按钮即可复制信息

### Cookie获取指引
如需获取完整Cookie用于其他工具，请按以下步骤操作：

1. **进入「我的收藏」页面**
   - 确保页面完全加载完成

2. **打开开发者工具**
   - 按 **F12** 键（或右键 → 检查）

3. **切换到「Network」标签**
   - 在开发者工具顶部选择「Network（网络）」标签页

4. **筛选请求**
   - 在筛选框中输入：`v1/web/aweme/listcollection`

5. **复制Cookie字段**
   - 点击任意一条筛选结果
   - 在右侧「Headers（标头）」中找到「Cookie」字段
   - **完整复制整段内容**（不可删减字符）

> **重要提示**：Cookie包含您的登录凭证，请妥善保管，避免泄露。

## 技术细节

### 版本信息
- **当前版本**：v10.2
- **更新日期**：2025年12月
- **主要变更**：移除自动Cookie获取，增加手动获取指引

### 工作原理
1. **网络请求监听**：通过重写XMLHttpRequest和Fetch API监听抖音数据请求
2. **数据解析**：从API响应中提取用户信息
3. **智能识别**：根据URL和页面特征判断主页类型
4. **自动触发**：在特定条件下自动点击页面元素触发数据加载

### 兼容性
- **浏览器**：Chrome, Firefox, Edge等现代浏览器
- **用户脚本管理器**：Tampermonkey, Violentmonkey
- **平台**：Windows, macOS, Linux, Android, iOS
